# Vue3 Mini Template

一个基于 Vue3、TypeScript、Vite6 的轻量级前端项目模板。

## 技术栈

- 🎯 Vue 3
- 🔧 TypeScript
- ⚡️ Vite 6
- 🎨 UnoCSS
- 🏬 Pinia
- 🛠️ VueRouter
- 📦 PNPM

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发环境启动
pnpm dev

# 测试环境启动
pnpm dev:test

# 生产环境启动
pnpm dev:pro

# 构建
pnpm build:dev    # 开发环境
pnpm build:test   # 测试环境
pnpm build:pro    # 生产环境
```

## 项目结构

```
├── src/
│   ├── assets/        # 静态资源
│   ├── components/    # 公共组件
│   ├── pages/        # 页面组件
│   ├── router/       # 路由配置
│   ├── store/        # 状态管理
│   ├── styles/       # 全局样式
│   └── types/        # TypeScript 类型
```
